English

If you choose to stay logged in, TagAlong will store the following personal data in your own device. These data are encrypted before storing.

  • Spotify User ID

These data will only be shared with Spotify in order to acquire certain information necessary for the app to function correctly.

If you decide you no longer desire your data to be stored, you just need to log out by navigating to the Settings screen and tapping Log out.

For any question regarding the storage and use of these personal data, you can contact me at mmaringomez.mmg@gmail.com.

Español

Si eliges mantener la sesión abierta, TagAlong almacenará la siguiente información personal en tu propio dispositivo. Esta información es encriptada antes de ser almacenada.

  • ID de usuario de Spotify

Esta información únicamente será compartida con Spotify para poder adquirir los datos necesarios para que la aplicación funcione correctamente.

Si decides que no deseas que tu información continúe almacenada, tan solo tienes que cerrar la sesión, navegando a la pantalla de Opciones y pulsando Cerrar sesión.

Para cualquier cuestión relacionada con el almacenamiento y uso de esta información personal, puedes contactar conmigo en mmaringomez.mmg@gmail.com.
